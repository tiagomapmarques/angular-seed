Angular Seed is a starter project that implements best practices in coding, building and testing Angular apps.

This project is based on the Angular-CLI template and inspired by Angular 2 Seed (a project by Minko Gechev).

Features

  • Ready to go, statically typed build system with TypeScript
  • Production and development builds
  • Sample unit and end-to-end tests
  • Development server with live reload
  • Example structure for a complete front-end app
  • Angular material sample usage
  • Versatile theming with Sass
